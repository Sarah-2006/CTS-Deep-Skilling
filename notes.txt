Task 1 - Angular Project File Explanations

1. angular.json
Configuration file for building and running the Angular project.

2. tsconfig.json
Contains TypeScript compiler settings.

3. tsconfig.app.json
TypeScript settings for the Angular application.

4. package.json
Stores project information, dependencies, and npm scripts.

5. src/main.ts
Entry point that starts the Angular application.

6. src/app/app.config.ts
Configures the application using the standalone API, including providers and routing.

7. src/app/app.component.ts
Root component of the Angular application.

8. src/index.html
Main HTML page where the Angular application is loaded.

Compiled application code is stored in main-FIPMWEQT.js inside the dist folder.
